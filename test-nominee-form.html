Test Purpose

This form tests the shortcode field that should appear in the Add Nominee modal. The field should be visible and allow both manual entry and auto-generation.

Unique code for USSD voting. Leave empty to auto-generate.
Troubleshooting

If shortcode field is not visible in the actual form:

  1. Check browser developer tools for JavaScript errors
  2. Verify the modal is not being hidden by CSS
  3. Ensure the form HTML is not commented out
  4. Check if Bootstrap CSS is loading properly
  5. Clear browser cache and refresh
Expected Behavior
  • Field Visibility: Shortcode field should be visible in the form
  • Auto-generation: If left empty, backend generates shortcode automatically
  • Validation: If entered manually, must match format like MHA012
  • Uniqueness: Backend checks for duplicate shortcodes
